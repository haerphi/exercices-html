Syntaxe markdown

Les listes

Non ordonnées

On précède chaque items par une astérisque "`*`".

  • Item 1
  • Item 2
    • Item 2a
    • Item 2b
  • Item 3

Ordonées

Pour imbriquer des listes, on indente la seconde liste dans la première.

  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3a
    2. Item 3b

Les liste de taches.

Imbrication

On imbrique des listes en les précédant par une indentation propre à chaque liste.

  • Item 1
  • Item 2
  • Item 1
  • Item 2
  • Item 3
  • Item 3

Images

Syntaxe à utiliser afin d'ajouter une image:
`![Alt](url)`
Une autre manière d'ajouter une image est:
`![Alt](/home/user/images/img.png)`

Titres :

Voici comment créer plusieurs titres : # Titre: équivaut à <h1> en html
## Titre: équivaut à <h2> en html
et ainsi de suite jusqu'à arriver au <h6> : ###### Titre.

Mise en forme :

  • Gras : ``**texte**``
  • Italique : ``*texte*``
    • ou `_texte_`
  • Gras & italique : ``***texte***``
  • Souligné : `__texte__`
  • Italique & souligné : `__*texte*__`
  • Gras & souligné : `__**texte**__`
  • Italique, gras & souligné : `__***texte***__`

Liens :

Il y a 2 façons de faire :

  • Soit en écrivant directement le lien en ayant pour nom l'url. *Ex :* http://github.com
  • Soit en donnant un nom au lien [nom du lien] qui, en cliquant dessus, vous dirigera vers l'url (url);[Github](http://github.com). Ex : Github

Code

Pour écrire du code en Markdown avec une coloration syntaxique il faut le code entre avec le nom du langage.

Exemple:
Pour faire ce ci: 

function fancyAlert(arg) {
            if(arg) {
                $.facebox({div:'#foo'})
            }
        }
        
        

            Il faut faire cela:
        

        ```javascript
            function fancyAlert(arg) {
                if(arg) {
                    $.facebox({div:'#foo'})
                }
            }
        ```
